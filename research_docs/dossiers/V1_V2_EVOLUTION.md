AEGIS-GRID DOSSIER: THE SHIFT TO INTELLIGENCE (V1 TO V2)

1. VERSION 1: THE REACTIVE SHIELD
What: A static Zero-Trust model with fixed thresholds.
Why: Designed for legacy grid protection where simple blocking was sufficient.
How: Rule-based logic (If Jitter > 50ms, then Alarm).
Result: Ineffective against slow-burn timing attacks and sophisticated traffic analysis.

2. VERSION 2: THE ADAPTIVE SUBSTRATE
What: Agent-centric security framework with 4-Vector Stochastic Reasoning.
Why: To combat adversaries who manipulate network physics without hitting fixed limits.
How: Implementation of the CAP Score (Criticality, Availability, Pressure). The agent uses pressure trends to predict attacks before they peak.
Key Change: Inverted the protocol. The agent now controls the encryption hardness and packet shuffling based on real-time trust vectors.
